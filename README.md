# 吾立方 Nuke 插件

## 使用

去[发布页面]下载压缩包

## 使用方法

- Windows

  运行 `安装.cmd` 后即可在 Nuke 中使用

## 开发

### 安装依赖

```shell
export PIPENV_VENV_IN_PROJECT=1
pipenv install -d --python "C:\Program Files\Nuke11.3v3\python.exe"
./scripts/setup.cmd
```

替换 `"C:\Program Files\Nuke11.3v3\python.exe"` 路径为你 Nuke 附带 python 的路径
