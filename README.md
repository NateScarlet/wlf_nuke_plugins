# 下载

去[发布页面](https://github.com/WuLiFang/Nuke/releases)下载压缩包

# 安装依赖

需要`python`和`pip`

运行`install_site_pacakages.bat`

如果不会就去[发布页面](https://github.com/WuLiFang/Nuke/releases)选择已附带依赖的包

# 使用方法

- Windows

  安装依赖后运行同目录下的`安装.bat`即可
